Planet Lab Balkanization Measurements

This project conducts a series of HTTP requests for a small number of common websites in order to characterize how frequently usesr are denied services based on their geolocation. Specifically, the measurement will retrieve the size of the main page resource, the size of the headers, and the HTTP status code.

The list of websites we are testing is here. Any questions can be sent to amcdon@umich.edu.
